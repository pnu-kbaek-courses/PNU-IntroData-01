## 과제 제출 방법

### 기본 방법: 로컬(개인 PC)에서 작업 후 GitHub 웹에서 업로드

1. `ch03_test.ipynb`를 다운로드하여 Jupyter Notebook에서 문제를 풀어주세요.
2. 완성된 `.ipynb` 파일을 준비합니다.
3. 이 repository의 `test/` 폴더로 이동합니다.
   - GitHub 웹에서 `test/` 폴더를 클릭한 후 **Add file → Upload files** 선택
   - 파일을 드래그 앤 드롭 하거나 선택하여 업로드
   - **Commit changes** 버튼을 눌러 저장 (이것이 곧 제출입니다)
4. 수업 종료 후 추가 제출이 필요한 경우, `test_additional/` 폴더를 동일한 방식으로 사용합니다.

> ⚠️ 이전에 제출한 파일이 있다면 동일한 파일명으로 업로드하면 자동으로 덮어씁니다. 마지막 commit 기준으로 제출본이 결정됩니다.
